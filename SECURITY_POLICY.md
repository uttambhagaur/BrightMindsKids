# Security Policy

Effective date: 10 January 2026

This Security Policy summarizes the technical and organizational measures Bhagaur maintains to protect personal and company data.

## 1. Governance
- Assigned security lead: foisonpro@gmail.com
- Regular risk assessments and policy reviews.

## 2. Access Control
- Principle of least privilege for systems and data access.
- MFA required for privileged accounts.

## 3. Data Protection
- Encryption in transit (TLS) and at rest where feasible.
- Regular backups and secure storage.

## 4. Infrastructure & Operations
- Hardened server configurations, patch management, and monitored logging.
- Use of vetted cloud providers and subprocessors.

## 5. Application Security
- Secure development practices, dependency management, and code reviews.
- Vulnerability scanning and periodic penetration testing.

## 6. Incident Response
- Incident response plan with detection, containment, eradication, recovery, and post-incident review steps.
- Incident notification: we will notify affected customers when required by law and contract.

## 7. Personnel & Training
- Background checks as allowed, security awareness training, and role-based training.

## 8. Third Parties
- Security requirements for vendors and subprocessors; we evaluate and monitor them.

## 9. Reporting
- Report security concerns to: foisonpro@gmail.com
